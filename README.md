# 📞 Keen Keeper 

## 📌 Short Description

Keen Keeper is a React-based friend relationship tracking application that helps users manage and maintain meaningful connections. Users can track calls, texts, and video interactions with friends and monitor relationship goals through an organized timeline.

---

## 🚀 Technologies Used

- React.js
- React Router
- Context API
- Tailwind CSS
- DaisyUI
- React Icons
- React Toastify
- JavaScript (ES6)

---

## ✨ Key Features

### 1. Friend Interaction Tracking
Users can log interactions like:
- Phone Call
- Text Message
- Video Call

All activities are automatically added to the Timeline.

---

### 2. Timeline Activity System
- Dynamic Timeline UI
- Activity icons (Call, Text, Video)
- Real-time updates using Context API

---

### 3. Relationship Goal Monitoring
Users can track:
- Days Since Contact
- Relationship Goal (Days)
- Next Due Date
- Friend Status (On-track / Almost Due)
